Tutoriel d'utilisation de Downloader

Ouverture du logiciel

1. La première ligne est pour la recherche de vidéo, insérez le lien puis cliquer sur "rechercher".

2. La deuxième ligne est pour définir l'emplacement dans lesquels vont être stockées les vidéos (À la fermeture du logiciel, celui-ci sauvegarde votre dossier en mémoire, afin de vous évitez de le redéfinir plus tard.)

Une fois que le lien à été recherché

1. Barre de progression indiquant l'avancement du téléchargement.

2. Qualité vidéo, vous permet de choisir une qualité vidéo disponible (par défaut sans choisir une qualité dans cette section, en appuyant sur "télécharger" la qualité vidéo sera de 720p)

3. Option MP3, cette option vous permettra de télécharger uniquement l'audio de la vidéo souhaitée

4. La file d'attente vous permet de visualiser le titre de toutes les vidéos en attente de téléchargement.

5. Bouton + et - vous permettent de gérer votre file d'attente, le bouton + enlèvera le lien afin que vous puissiez en rechercher un autre sans attendre.

6. Bouton télécharger vous permet de lancer le téléchargement, une fois terminé, retrouvez la/les vidéo(s) dans le dossier précédemment choisis !

Choses à savoir

File d'attente: Si vous voulez télécharger une vidéo en 1080p et ensuite l'audio d'une musique, vous pouvez. Il vous suffira de sélectionner les éléments approprié. Pour reprendre l'exemple pour une vidéo 1080p on sélectionne 1080p dans la liste et appuie sur "+" ensuite pour l'audio d'une musique à la suite, on coche "MP3" et ensuite "+"(N'oubliez pas de décocher "MP3" si la vidéo d'après vous ne la voulez pas en audio seulement). Enfin, appuyez sur télécharger et vous aurez télécharger 2 types de vidéos différents

Liste de qualité vidéo: Lorsque vous sélectionner une qualité vidéo, le logiciel téléchargera la vidéo et l'audio séparemment et les combinera par la suite. Si vous avez l'utilité de vouloir garder la piste vidéo uniquement, vous pouvez aller dans le dossier ou est installé le logiciel, dans le fichier "settings.json", ouvrez le avec un bloc-note et changer le paramètre "Supprimer_fichier_resolution", mettez "Non" à la place de "Oui"

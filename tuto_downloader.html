Tutoriel d'utilisation de Downloader

1. La première ligne est pour la recherche de vidéo, insérez le lien puis cliquer sur "rechercher"

2. La deuxième ligne est pour définir l'emplacement dans lesquels vont être stockées les vidéos (a la fermeture du logiciel, celui-ci sauvegarde votre dossier en mémoire, afin de vous évitez de le redéfinir plus tard)

1. Barre de progression indiquant l'avancement du téléchargement.

2. Qualité vidéo, vous permet de choisir une qualité vidéo disponible (par défaut sans choisir une qualité dans cette section, en appuyant sur "télécharger" la qualité vidéo sera de 720p)

3. Option MP3, cette option vous permettra de télécharger uniquement l'audio de la vidéo souhaiter

4. La file d'attente vous permet de visualiser toutes les vidéos que vous allez télécharger.

5. Bouton + et - vous permettant de gérer votre file d'attente, le bouton + enlèvera le lien afin que vous puissiez en rechercher un autre sans attendre.

6. Bouton télécharger vous permet de lancer le téléchargement.
